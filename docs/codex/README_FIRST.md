# Simple Case Portal - Codex Implementation Pack

This pack is prepared to help Codex implement the **Simple Case Portal** application using:

- ASP.NET Core MVC
- SQL Server
- Entity Framework Core Code First
- Layered Architecture
- DigitalOcean Spaces / S3-compatible storage
- Manual API sync to Dentist App, Workflow App, and Production App
- High-security long-term file retention up to 10 years

## Recommended Usage

Place these files inside the repository under:

```txt
/docs/codex/
```

Then open `CODEX_MASTER_PROMPT.md` and paste it to Codex.

## Files Included

| File | Purpose |
|---|---|
| `CODEX_MASTER_PROMPT.md` | Main full prompt to give Codex |
| `MEMORY_TREE.md` | Stable project memory and business rules |
| `PROPOSED_FILE_TREE.md` | Suggested .NET MVC solution structure |
| `DATABASE_BLUEPRINT.md` | Initial entities and database tables |
| `IMPLEMENTATION_TASKS.md` | Step-by-step implementation phases |
| `SECURITY_REQUIREMENTS.md` | Security, storage, audit, backup, and 10-year retention requirements |
| `API_SYNC_CONTRACTS.md` | Manual API sync payloads and response design |
| `COST_ESTIMATE.md` | Development and running cost section |
| `CODEX_COPY_PASTE_SINGLE_PROMPT.txt` | One combined prompt if you prefer a single prompt |
